# Voltage-Stabilizer-With-Overcurrent-Protection

## Project Overview
This project presents the design and simulation of an analog voltage stabilizer with integrated overcurrent protection. The circuit is designed to provide a stable output voltage between **7 V** and **20 V** when powered from an input supply of **25 V**, while ensuring safe operation through a **maximum output current limit of 1.5 A**.

---

## Features

- Maximum output current: **1.5 A**
- Integrated **overcurrent protection**
- Stable output voltage across varying load conditions and temperature
- Thermal behavior analyzed for **0°C – 40°C temperature**
- **Practical hardware implementation**

---

## Simulation

Simulations were performed in **OrCAD** to evaluate the behavior of the stabilizer under multiple operating scenarios:

- Load variation across different resistance values with tolerance
- Output current approaching the **1.5 A limit**
- Temperature variation between **0°C and 40°C**

The simulation results confirm stable voltage and proper activation of the overcurrent protection mechanism for tested conditions.
